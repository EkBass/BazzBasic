1.

- Download BazzBasic binary from Google Drive.

- Extract the ".zip" package to a folder of your choice.

- If your firewall warns you about BazzBasic, you can ignore it. You may need to set an exception in some cases.

2.

- Browse to folder where you downloaded bazzbasic.exe

- Double-click bazzbasic.exe and IDE opens.

- Study subfolder Examples to start the fun.

- Study BazzBasic Wiki to learn more.

3.

- The integrated IDE which comes with BazzBasic is very basic and simple.

- But it allows you to let your curiosity be awakened in the easiest way possible.

- BazzBasic syntax highlighting is also available for notepad++, Geany and Visual Studio Code.

- Read more at here

ISSUES WITH FIREWALL?

While developing BazzBasic, I have noticed that Avast Security at least scans it almost every time I have compiled the BazzBasic source code into a binary.

Don't worry though. The BazzBasic source code is available to everyone and you can check for yourself that it doesn't do anything illegal on your machine. In time, Avast learn that BazzBasic is not malware, but it takes a while.
