# More Command Line Features


## Get a link to the BazzBasic Beginner's Guide

```
bazzbasic.exe -guide
```

**or**

```
bazzbasic.exe -help
```

## Get Version of Your Current BazzBasic

```
bazzbasic.exe -version
```

**or**

```
bazzbasic.exe -v
```

## Check For New Version Release

```
bazzbasic.exe -checkupdates
```
