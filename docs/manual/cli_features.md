# More Command Line Features


## Get a link to the BazzBasic Starter's Guide

```
bazzbasic.exe -guide
```

**or**

```
bazzbasic.exe -help
```

## Get Version of Your Current BazzBasic

```
BazzBasic.exe -version
```

**or**

```
BazzBasic.exe -v
```

## Check For New Version Release

```
BazzBasic.exe -checkupdate
```
