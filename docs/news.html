18th Jan 2026

BazzBasic is now in version 0.5c
It can already be used for many things, but it still lacks a few particularly important things that I would like to have released with version 0.6.

  • Bundling the BASIC code and BazzBasic together. After this, you can distribute your code as an ".exe" file.
  • ALPHA color. At the moment, the so-called "green screen" technique is not possible.
  • POINT x, y. That is, reading the color from a certain coordinate on a graphic display is not possible.

The goal is for version 0.6 to be released during January 2026.
